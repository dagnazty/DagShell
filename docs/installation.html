  • Orbic RCL400 LTE Hotspot (with root access)
  • Computer with USB connection to the hotspot (or WiFi)

  • ARM Cross-Compiler - arm-none-linux-gnueabihf-gcc (from ARM GNU Toolchain)
  • Python 3 - For running deployment scripts
  • PowerShell - For Windows build script

üî® Building from Source

Clone the Repository

git clone https://github.com/dagnazty/DagShell.git
cd DagShell

Install ARM Toolchain

Download the ARM GNU Toolchain from ARM Developer.

Make sure arm-none-linux-gnueabihf-gcc is in your PATH, or edit the path in build.ps1.

Build the Firmware

cd orbic_fw_c
.\build.ps1

This produces orbic_app - a statically-linked ARM binary (~200KB).

üöÄ Deployment

There are two methods to deploy DagShell to your device:

Method 1: Base64 Transfer (Recommended)

This method encodes the binary to base64 and sends it in chunks via the diagnostic port.

Connect to Hotspot WiFi

Connect your computer to the Orbic's WiFi network.

Edit Script Paths

Open deploy_base64.py and update FIRMWARE_DIR to point to your local build.

Run Deploy Script

python deploy_base64.py

The script will:

  • Kill any existing DagShell process
  • Transfer the binary in chunks
  • Decode and make executable
  • Start the HTTP server

Method 2: Network Transfer

This method hosts the binary on a local HTTP server and downloads it via wget.

Configure

Edit deploy_net.py and set HOST to your PC's IP on the RNDIS interface.

Run Deploy Script

python deploy_net.py

This starts a local HTTP server and instructs the device to download the firmware.

üîß Troubleshooting

Connection Refused on Port 24

The diagnostic port may not be open. Ensure your root exploit properly exposes port 24.

Binary Not Executing

Make sure the binary was compiled statically for ARM. Check with file orbic_app - it should show ARM architecture.

"Modem Port Busy" Errors

Another process may be using /dev/smd8. The firmware will retry automatically, but if persistent, reboot the device.

Page Not Loading

Verify DagShell is running by connecting to port 24 and running ps | grep orbic.
